Main Menu       ~ Main page
    New Game        - opens New Game
    Continue        - opens the file explorer, letting the player open a .puzzle file. 

New Game        ~ Page where the user configures the puzzle
    Select Image    - opens the file explorer, letting the player open an image used as the puzzle. 
    Size            - dropdown where the player choses roughly how many pieces
    Use Rotation    - toggles if rotation is used in the puzzle. '
    Play            - opens Game Window

Game Window     ~ The actual puzzle board
    Save Game
    Main Menu

There is also a button "play game" It opens a new screen with the a jumbled pile of pieces. 

Buttons at the top: 
 - Button to space them out around the puzzle board. 
 - Button to only view edges. 
 - Button to toggle finished image overlay. 
 - Button to bring up image. 

The user can drag each piece with left mouse drag. When pressing on a piece it rotates clock wise. 

When dragging a peace to its neighbour they connect and act like one island. Movements, rotations 
are performed for the entire island.
